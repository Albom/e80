# e80

A ZX Spectrum emulator for KolibriOS.
&copy; 2008-2020 Albom 

e80 is based on Z80 emulator by Santiago Romero Iglesias.

## Keys
|  ZX Spectrum |PC   |
| :------------ | :------------ |
|CS (CAPS SHIFT)   |   Shift|
| SS (SYMBOL SHIFT)  | Ctrl  |
|  CS+SS | Alt  |

| Key  | Action  |
| :------------ | :------------ |
|   F12 | Reset   |
|   F2 |   Save snapshot  (.sna)|
|   F3 |   Load snapshot (.sna)|
|   F2 |   Save screen  (.scr)|
